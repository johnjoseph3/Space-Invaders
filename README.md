# Space Invaders

ES6 Space Invaders written with the [Canvas API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API).

### Development

Run webpack-dev-server on port 8080:

```sh
$ npm run start-dev
```

### Production
Run production on port 4000:

```sh
$ webpack
$ npm run start-prod
```
### Tech
Webpack, LESS, and Express.
